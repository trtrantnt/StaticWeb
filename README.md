# Website 9 anh (1-9)

Trang web hien thi 9 anh (`1.jpg` den `9.jpg`) theo dang gallery, giao dien de nhin va tuong thich mobile.

## Chay local

Chi can mo file `index.html` bang trinh duyet.

## Host len GitHub Pages

1. Tao repo moi tren GitHub, vi du: `vui-gallery`.
2. Upload toan bo file trong thu muc nay len nhanh `main`:
   - `index.html`
   - `1.jpg` -> `9.jpg`
   - `README.md`
3. Vao **Settings** -> **Pages**.
4. Tai **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Nhan **Save**.
6. Cho 1-2 phut, website se co link dang:
   - `https://<github-username>.github.io/vui-gallery/`

## Neu anh khong hien thi

- Kiem tra ten file phai dung chinh xac: `1.jpg` ... `9.jpg`.
- Kiem tra file nam cung cap voi `index.html`.
- Neu doi ten repo, cap nhat lai URL Pages tuong ung.
